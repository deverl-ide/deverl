toggle_menu_group/2

toggle_menu_group(Mask::integer(), Toggle::boolean()) -> ok

Enable/disable a menu group. Menu items can be associated to one or more menu groups (bit flags, defined in ide.hrl). The Mask is compared using bitwise AND to every menu item. If the menu item has the group bit set then it will be enabled when Toggle is true and disabled when Toggle is false.
